Larasonic makes sitemap generation simple using [spatie/laravel-sitemap](https://github.com/spatie/laravel-sitemap).

## ğŸ”„ Generate Sitemap

```bash
php artisan generate:sitemap
```

This creates `sitemap.xml` in your public folder using your `APP_URL`.

To generate a production sitemap locally, modify the `$url` variable in `app/Console/Commands/GenerateSitemap.php`.

<Card>
Larasonic automatically schedules daily sitemap generation in `routes/console.php`:

```php:routes/console.php
// Automatically generate sitemap daily at midnight
Schedule::command('generate:sitemap')->daily();
```
